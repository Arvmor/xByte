# Quick Actions

---

## Installation

Install the xByte SDK using your preferred package manager:

:::code-group

```bash [bun]
bun add xbyte-sdk
```

```bash [npm]
npm install xbyte-sdk
```

```bash [pnpm]
pnpm add xbyte-sdk
```

:::

## API Client

The [`xByteClient`](/api-client) provides REST API access to the xByte backend for managing clients, buckets, pricing, and content.

:::code-group

```typescript [Create a client]
import { xByteClient } from "xbyte-sdk";

const client = new xByteClient();

const response = await client.createClient({
  name: "My Content Platform",
  wallet: "0x1234...",
});
```

```typescript [Set pricing]
const pricingResponse = await client.setPricing({
  clientId: "your-client-id",
  pricePerMb: "1000000",
});
```

```typescript [Get content]
const content = await client.getContent({
  clientId: "your-client-id",
  objectKey: "video.mp4",
  offset: 0,
  length: 1048576,
});
```

:::

## EVM Client

The [`xByteEvmClient`](/evm-client) enables direct interaction with xByte smart contracts on-chain.

:::code-group

```typescript [Initialize client]
import { xByteEvmClient } from "xbyte-sdk";

const evmClient = new xByteEvmClient("https://sepolia.base.org");
```

```typescript [Compute vault address]
const vaultAddress = await evmClient.computeVaultAddress(
  "0x1234567890123456789012345678901234567890"
);
```

```typescript [Get balance]
const balance = await evmClient.getBalance(vaultAddress);
```

:::
